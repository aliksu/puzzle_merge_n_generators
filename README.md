# Puzzle: merge N generators' values into single

##### Input:
N generators with sorted values

##### Output:
Single generator with sorted values

***

#### Example
Input:
```
[1, 2, 4, 5]                                  
[1, 3, 4, 4, 7]
[1, 1, 2, 3]
```

Output:
```
[1, 1, 1, 1, 2, 2, 3, 3, 4, 4, 4, 5, 7]
```
